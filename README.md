# ALOHA ATM
This is the documentation for our praktikum and seminar "Deep Learning in Robotics".

## Setup
- clone the repo
- cd aloha_atm
- conda create -f environment.yml
- conda activate aloha_atm
- python3 scripts/preprocess_aloha.py
